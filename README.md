# React Toolchain

A custom toolchain to study, understand and practice what React toolchains and frameworks makes behind of they stages.



### Instructions

- To install

  ```bash
  npm install
  ```
- To execute

  ```bash
  npm start
  ```


The app should execute and show the result at http://localhost:3000
